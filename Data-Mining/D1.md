# Day 01 Notes: Data Mining & Models

Today, I learned some new concepts about Data Mining, models, and the tools used in this field.

**Data Mining**: Making predictions on new data based on past data.  
**Tools**: Models.

I learned about two main types of models: **Supervised** and **Unsupervised**.

### Supervised
- Has both inputs (features) and outputs (labels).
- Goal: Predict the correct answer for new data based on what the model has learned.
- Examples: Classification, Regression.

### Unsupervised
- Has only inputs (features), with no labels.
- Goal: Find patterns, similarities, or differences in the data.
- Example: Clustering.
